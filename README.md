# RID221037_Desafio1
Exercício: Desenvolvendo uma Landing Page - Plataforma DNC

## O que foi codificado

### **index.html**
- Estrutura semântica HTML com menu de apresentação (título + subtítulo)
- Seção informacional com 3 colunas de estatísticas (850 empreendimentos, 40 anos, 2.000.000 m²)
- Seção de conteúdo com texto descritivo e imagem da sala
- Formulário de contato integrado com SheetMonkey (campos: nome, email, data automática)

### **style.css**
- Layout fixo de 1440px com design profissional
- **Grid Layout**: Menu informacional (3 colunas) e conteúdo principal (2 colunas)
- **Flexbox**: Centralização do formulário e elementos
- Imagem de fundo SVG no cabeçalho
- Estilização dos inputs e botão com bordas arredondadas
- Efeitos hover no botão (elevação e sombra)

### **index.js**
- Arquivo preparado para futuras interações JavaScript

### **Imagens**
- `image_sala.png` - Foto de interiores
- `retangulo.svg` / `retangulo10.png` - Elementos visuais de design
- `relangulo20.svg` - Componente gráfico adicional
